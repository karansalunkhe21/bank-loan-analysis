This Power BI project provides comprehensive analytics and insights into bank loan portfolios, enabling data-driven decision-making for loan approval, risk assessment, and portfolio management. Key Features

Loan Performance Tracking: Monitor loan status (Fully Paid, Charged Off, Current)
Risk Assessment Dashboard: Analyze loans by grade, sub-grade, and credit metrics
Geographic Analysis: State-wise loan distribution and performance
Borrower Profiling: Employment length, home ownership, and income verification analysis
Financial Metrics: DTI ratios, interest rates, installment tracking, and total payment analysis
Time-Series Analysis: Loan issuance trends, payment patterns, and aging analysis
